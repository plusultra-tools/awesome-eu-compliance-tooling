# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in full.

The complete and authoritative text of the Code of Conduct is published at:

- HTML: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Markdown: https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md
- Translations: https://www.contributor-covenant.org/translations

By participating in this project — opening issues, submitting pull requests, commenting on threads, or any other interaction — you agree to abide by the Contributor Covenant 2.1 as published at the URLs above.

## Reporting

Instances of behavior that violates the Contributor Covenant may be reported to the project maintainers at:

```
plusultra.dev@proton.me
```

Reports are handled confidentially. The maintainer team will acknowledge receipt within 7 days and follow the enforcement guidelines section of the Contributor Covenant 2.1.

## Scope

This Code of Conduct applies within all project spaces (repository, issues, pull requests, discussions, project-affiliated communication channels) and in public spaces when an individual is officially representing the project.
